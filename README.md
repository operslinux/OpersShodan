# [Instalacion]
```
$ apt update && apt upgrade
```
```
$ apt install git python3
```
```
$ git clone https://github.com/operslinux/Opers-Shodan.git
```
```
$ cd Opers-Shodan
```
```
$ sudo chmod +x run.py
```

# [ Setup ]
```
$ pip3 install -r requirements.txt
```
# [ Running ]
```
$ python3 run.py
```
